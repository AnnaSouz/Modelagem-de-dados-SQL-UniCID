# Modelagem-de-dados-SQL-UniCID

Este repositório contém scripts SQL completos baseados no modelo lógico criado para o minimundo escolhido, incluindo criação de tabelas, inserção de dados, consultas e manipulações (UPDATE e DELETE).

#Arquivos:
create_tables.sql – Criação do banco com todas as PKs e FKs
insert_data.sql – Populamento select_queries.sql – Consultas com WHERE, ORDER BY, JOIN, GROUP BY e LIMIT
update_delete.sql – Três comandos UPDATE e três DELETE

#Executar:
1. Crie o banco:
CREATE DATABASE floricultura;
USE floricultura;

3. Execute os scripts na ordem:

create_tables.sql

insert_data.sql

select_queries.sql

update_delete.sql
